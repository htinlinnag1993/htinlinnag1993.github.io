Journey

------------------------------

  • June-August 2016

    Wow Moment @ BotFactory

    During the software & hardware engineering internship at BotFactory, while working on 3D printers & their softwares, with the support of my supervisors, I discovered my strong passionate for Software Engineering.

  • September 2016

    The Shift in My Pursuit

    My urge to learn more about Computer Science took me to Queens College to begin my pursuit in CS. My background in Electrical Engineering helped me understand CS in a whole new different level. Graduating in December 2018.

  • January-May 2017

    Acceleration Begins

    TTP Residency @ NYC Tech Talent Pipeline. Completed a highly selective program (Acceptance Rate is less than 15%) delivered by the New York Code + Design Academy for Full-Stack Web Development & Soft Skills in a 25-student-team.

  • February-June, 2017

    Engineer at Work

    Software Engineering Intern at Neuromatters. Assisted & developed VR games, integrating Oculus Rift with Brain Computer Interface Technologies (EEG). Designed & built a client-facing data-visualization web player app to analyze brain data.

  • April-May 2018

    Java & SCRUM

    Software Engineer & Scrum Master
    at MEDefine. Managed a team of 4 developers for 2 intensive sprints. Led series of user interviews & clarified user stories. Initiated & arranged proper system design. Built a secure user authentication system with SSL encrypted remote database connection using Java Servlets, MySQL, BCrypt, AWS & Apache Tomcat.

  • July-August, 2018

    Proper Practices & Theories

    Internet & Web Technologies class at QC. Learned & experimented with various internet protocols, architectures, designs, DOM manipulation, webscraping, data exchange, AJAX, JavaScript, NodeJS, SQL Databases, NoSQL Databases, mobile web development, APIs, etc.

  • June-August, 2018

    Emaily & New Framework

    Web App for Creating Email Surveys. Explored, learned to use & implemented new libraries(React, Redux, PassportJS, Mongoose), new APIs(Google OAuth 2.0, Stripe, SendGrid), MongoDB, mLab & Heroku

  • August, 2018-Present

    Split & Remember

    Web App for Splitting Bills & Remembering What Friends Owe to Each Other. Built by using HTML5, CSS5, React, Redux, NodeJS, D3, PassportJS, Babel, Google OAuth 2.0, SendGrid, MongoDB, mLab & Heroku.

  • August-December, 2018

    Excited!!!

    My last semester in Queens College. Learning new topics: Algorithms for Big Data, Image Processing, Computer Graphics, Principles of Programming Languages & Biometric Security & Privacy. Submitting job applications & preparing for coding interviews.

  • Life's a journey

    My curiousity never ends!

    Neither does learning & sharing...
